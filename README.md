# Приложение Заметки (NotesApp)

Это простое приложение для заметок, сделанное по урокам на YouTube.

## Функционал

- Создание заметок
- Редактирование заметок
- Удаление заметок
- Сохранение заметок

## Используемые технологии

- Android SDK
- Kotlin
- RecyclerView
- SearchView
